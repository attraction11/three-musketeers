我是标题文字内容

代码

  • HTML：
    <div class="title">我是标题文字内容</div>
  • CSS：
    .title {
  width: 8em;
  margin: auto;
  white-space: nowrap;
  animation: textIn 1s both;
  letter-spacing: -200px;
}
@keyframes textIn {
  0% {
    opacity: 0;
    letter-spacing: -200px;
  }
  60% {
    letter-spacing: 5px;
  }
  100% {
    opacity: 1;
    letter-spacing: 0;
  }
}
